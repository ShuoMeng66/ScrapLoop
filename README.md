# 绿环纪念站

公益回收主题网页游戏原型。玩家在回收基地中选择关卡，收集废物碎片、升级装备、合成纪念品，并通过 Boss 掉落的打印碎片兑换 3D 废物打印实物券。

## 本地运行

```bash
python -m http.server 4177 --bind 127.0.0.1
```

打开：

```text
http://127.0.0.1:4177/
```

## 玩法模式

1. 小怪清理：击败污染机械，获得塑料、金属、电路、电池、齿轮等废物碎片，用于升级装备和合成普通纪念品。
2. Boss 模式：挑战蓝猫回收守卫。Boss 掉落打印碎片，可兑换 3D 废物打印实物券。
3. 尸潮防守：连续防守 60 秒或击败足够敌人。敌人会持续涌入，场地内定时刷新 buff。
4. 解密回收：按提示收集材料，并按顺序投放到分拣设备。完成后获得回收章和电子类材料。

## 功能

- 主页设施：传送门、装备间、纪念品工坊、实物兑换台、任务板、修复站、材料精炼机、背景终端。
- 战斗系统：圆圈震荡、前方挥砍、追踪子弹、混合攻击等武器类型。
- 装备系统：武器、头盔、护甲、鞋子可解锁、穿戴、脱下，角色和装备界面会同步显示穿戴效果。
- Buff 系统：修复、护盾、加速、磁力、强化、冷却等 buff 在战斗场地内定时刷新；怪物和 Boss 不掉落 buff。
- 兑换系统：普通纪念品兑换公益点，Boss 打印碎片兑换 3D 打印实物券。
- 背景系统：使用碎片解锁主题背景，并在主页与弹窗界面中展示。

## 操作

- WASD / 方向键：移动
- Space / 点击画面：攻击
- Enter：互动、进入传送门、投放解密材料、回主页
- E：装备间
- C：收藏品合成
- X：实物兑换
- B：背景终端
- M：公益任务板
- N：通关小怪或尸潮后继续下一轮
- Esc：关闭面板，或退出关卡回主页安全区

## 建议打法

- 初期先刷小怪清理，积累塑料、金属、齿轮和回收章。
- 近战挥砍适合打 Boss，追踪子弹适合边走位边输出。
- 尸潮模式优先拾取场地 buff，磁力和冷却可以显著提升容错。
- 解密模式看底部提示，必须按当前目标材料的顺序投放。
- Boss 三阶段弹幕更密集，注意绕圈走位，利用护盾和修复站提高容错。

## 公网发布

项目是纯静态网页，所有资源都使用相对路径，可以部署到任意静态托管服务。

Netlify Drop：

1. 打开 https://app.netlify.com/drop
2. 上传整个 `recycle-quest` 文件夹内容，或上传准备好的 `recycle-quest-netlify.zip` 解压后的文件夹。
3. Netlify 生成公网 URL 后，任何电脑都可以直接打开游玩。

GitHub Pages：

1. 在 GitHub 新建一个公开仓库。
2. 上传 `index.html`、`main.js`、`style.css`、`README.md` 和整个 `assets/` 文件夹。
3. 在仓库 `Settings -> Pages` 中选择 `main` 分支和根目录发布。
4. 等待 GitHub Pages 生成公网 URL。
